# shop-kardi

Монорепозиторий нового сайта `shop.kardi.ru`.

## Стек

- `frontend`: Next.js, React, TypeScript, Tailwind CSS
- `backend`: Strapi, PostgreSQL

## Структура

```text
.
├── frontend
├── backend
├── docs
└── shared
```

## План реализации

1. Собрать витрину магазина на Next.js.
2. Поднять CMS и административную часть на Strapi.
3. Описать модели данных, SEO, формы, корзину и checkout.
4. Подготовить миграцию контента с Битрикс.

## Статус

В репозитории создается стартовый каркас под реализацию по ТЗ.
