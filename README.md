# shop-kardi

Монорепозиторий нового сайта `shop.kardi.ru`.

## Стек

- `frontend`: Next.js, React, TypeScript, Tailwind CSS
- `backend`: Strapi, PostgreSQL

## Структура

```text
.
├── frontend
├── backend
├── docs
└── shared
```

## План реализации

1. Собрать витрину магазина на Next.js.
2. Поднять CMS и административную часть на Strapi.
3. Описать модели данных, формы, корзину и оформление заказа.
4. Подготовить SEO-слой и миграцию контента с Битрикс.

## Статус

Базовая версия сайта работает локально:
- каталог и карточка товара
- корзина и оформление заказа
- отправка заявок (Lead)
- Strapi + PostgreSQL
