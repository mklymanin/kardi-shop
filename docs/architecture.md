# Архитектура

## Приложения

### frontend

Публичный сайт магазина на Next.js:

- главная
- каталог
- карточка товара
- корзина
- оформление заказа
- статьи и статические страницы
- поиск

### backend

CMS и API на Strapi:

- товары
- категории
- страницы
- статьи
- лиды
- заказы
- SEO-настройки
- общие настройки сайта

## Взаимодействие

- `frontend` получает данные из `Strapi` по REST API или GraphQL
- `backend` хранит данные в `PostgreSQL`
- загрузка изображений идет через Strapi Upload plugin
- формы и заказы сохраняются в Strapi
